Aplikasi Bank BGA adalah program sederhana berbasis C++ dan Qt Framework yang digunakan untuk mengelola tabungan, mengirim uang, serta mengatur daftar kontak. Proyek ini dibuat sebagai tugas Pemrograman Dasar Semester Gasal 2025/2026.
Aplikasi tersedia dalam dua versi:
1. Console UI (C++)
2. GUI (Graphical User Interface) menggunakan Qt Framework
